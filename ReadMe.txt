*Sample Assignment for Tester Role in Funda:*
********************************************
> The framework is build using Selenium - Maven - Java - Junit Framework. 
  To execute the test you would need to open this in Eclipse.

Structure of the Frame Work:
---------------------------
The tests can be find in the folder src/test/java
> features - contains all the test cases and the test cases which can be automted are tagged as "@Regression"
> StepDefinitios - contains the technical steps that need to be performed for the automatedtest cases
> pageObjects - Contains all the elements in the FundaNL page.
